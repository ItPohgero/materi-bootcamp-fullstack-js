## Learning Path

### Level 1: Foundation (Wajib untuk Semua)

#### 01. Git - Version Control System
**Durasi:** 2-3 hari  
**Prasyarat:** Tidak ada

Pelajari version control untuk manage code dan collaborate dengan tim.

**Topik:**
- Pengenalan Git dan konsep dasar
- Instalasi dan konfigurasi
- Repository management (init, clone)
- Basic workflow (add, commit, push, pull)
- Branching dan merging
- Remote repository
- Membatalkan perubahan
- .gitignore

**Mengapa Pertama?**
- Essential untuk semua development work
- Protect your code
- Collaboration fundamental
- Industry standard

**Output:** Mampu manage code dengan Git dan GitHub

---

### Level 2: Programming Fundamentals

#### 02. JavaScript - Programming Language
**Durasi:** 2-3 minggu  
**Prasyarat:** Git basics

Pelajari JavaScript sebagai bahasa pemrograman dengan fokus ke backend development menggunakan Bun.js.

**Topik:**
- Pengenalan JavaScript dan Bun
- Setup development environment
- Variabel dan tipe data
- Operators
- Control flow (if, switch)
- Loops
- Functions
- Arrays
- Objects
- File system operations
- HTTP server dengan Bun

**Mengapa Setelah Git?**
- Butuh Git untuk practice dan save progress
- JavaScript adalah bahasa yang versatile
- Bun.js modern dan sangat cepat
- Foundation untuk TypeScript

**Output:** Mampu build backend applications dengan JavaScript dan Bun

---

### Level 3: Type-Safe Development

#### 03. TypeScript - Type-Safe JavaScript
**Durasi:** 1-2 minggu  
**Prasyarat:** JavaScript fundamentals

Pelajari TypeScript untuk membuat JavaScript code lebih safe dan maintainable.

**Topik:**
- Pengenalan TypeScript
- Setup TypeScript dengan Bun
- Basic types dan type system
- Interfaces dan Type Aliases
- Functions dengan types
- Classes dan OOP
- Generics
- Advanced types
- Modules
- TypeScript dengan Bun APIs

**Mengapa Setelah JavaScript?**
- Build on top of JavaScript knowledge
- TypeScript adalah JavaScript + Types
- Essential untuk production applications
- Better tooling dan IntelliSense

**Output:** Mampu build type-safe applications dengan TypeScript

---

### Level 4: Data Management

#### 04. SQL (PostgreSQL) - Database
**Durasi:** 2-3 minggu  
**Prasyarat:** JavaScript basics (untuk integration)

Pelajari SQL untuk manage dan query data menggunakan PostgreSQL.

**Topik:**
- Pengenalan SQL dan PostgreSQL
- Instalasi dan konfigurasi
- Database dan Table management
- Tipe data PostgreSQL
- Query data dengan SELECT
- Filtering dengan WHERE
- Sorting dan limiting
- Insert, Update, Delete
- JOIN tables
- Aggregate functions dan GROUP BY

**Mengapa Terakhir?**
- Applications butuh database untuk store data
- SQL independent tapi integrate dengan JS/TS
- Complete backend development stack

**Output:** Mampu design database dan write complex queries

## Recommended Learning Schedule

- Setup GitHub account

**Week 3-5: JavaScript**
- Basic syntax dan concepts

**Week 6-8: JavaScript Advanced**
- File system operations
- HTTP servers dengan Bun
- Build REST API
- Testing

**Week 9-10: TypeScript**
- Type system basics
- Interfaces dan generics
- Convert JavaScript projects ke TypeScript

**Week 11-14: SQL**

- Query fundamentals
- JOIN operations

**Week 2-4: JavaScript + TypeScript** (combined)

## How to Use This Material

### For Complete Beginners:

## Study Tips

### 1. Practice, Practice, Practice
Coding adalah skill yang harus dipraktikkan. Jangan hanya baca, tapi code!

### 2. Build Projects
Apply apa yang dipelajari ke real projects:
- Todo app
- Blog API
- E-commerce backend
- CLI tools

### 3. Use Git dari Awal
Commit setiap progress, branch untuk experiments.

### 4. Read Error Messages
Error messages adalah teachers terbaik. Baca dengan teliti.

### 5. Debug Effectively
Use console.log, debugger, dan understand call stack.

### 6. Join Communities
- Stack Overflow
- Reddit (r/learnprogramming, r/javascript)
- Discord servers (Bun, TypeScript)
- Local meetups
